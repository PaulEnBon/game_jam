# 📦 Betrayal Box - Nothing is Safe

**Betrayal Box** est un FPS de survie frénétique qui fusionne l'esthétique des classiques du raycasting (Doom/Wolfenstein) avec un univers de voxels moderne. Coincé dans une arène qui se réduit inexorablement, vous devrez faire face à des vagues de zombies mutants tout en gérant vos ressources et votre arsenal.

## 🌌 L'Univers
Vous êtes piégé dans une dimension instable. Le sol sous vos pieds n'est pas votre allié : une corruption mystérieuse, se manifestant sous forme de lave brûlante, dévore les bords de l'arène à intervalles réguliers. Les "Trahis" (Betrayers), des créatures autrefois humaines, rodent dans les ombres pour vous empêcher de voir le lever du soleil. Dans la Betrayal Box, la seule constante est le changement... et le danger.

## ⚙️ Principes Fondamentaux
- **Survie par Vagues** : Les ennemis deviennent plus nombreux, plus résistants et plus rapides à chaque vague.
- **La Corruption (Battle Royale Style)** : Toutes les 35 secondes, une nouvelle couche de lave recouvre les bords de la carte. L'espace de combat se réduit, vous forçant à affronter les monstres de plus près.
- **Économie de Guerre** : Éliminez des ennemis pour récolter de l'**Or**. Cet or est vital pour améliorer votre équipement via la mystérieuse **Punch Machine**.
- **Mouvement et Énergie** : Utilisez votre sprint pour échapper aux situations désespérées, mais surveillez votre barre d'énergie.

## 🛠️ Fonctionnalités

### 🔄 Système de Double Rendu Unique
Basculez à tout moment entre deux époques du jeu vidéo :
- **Mode 2D Raycaster** : Un rendu logiciel pur, inspiré des FPS des années 90, avec des sprites pixelisés et une interface rétro agrandie.
- **Mode 3D Voxel** : Un monde en relief style Minecraft avec des modèles de zombies en 3D et des éclairages dynamiques.
*Appuyez sur **'O'** pour changer de perspective en plein combat.*

### 🔫 Arsenal et Équipement
1. **Pistolet de Service** : Votre arme principale. Améliorez son niveau pour augmenter drastiquement ses dégâts et la taille de son chargeur.
2. **Bombes à Fragmentation** : Idéales pour le contrôle de foule. Maintenez le clic droit pour visualiser la trajectoire avant de lancer.
3. **Noyaux de Pulse** : Déclenchez une onde de choc énergétique qui repousse tous les ennemis proches, vous offrant une seconde de répit.

### 🎰 La Punch Machine
Située au cœur de l'arène, cette machine est votre seul espoir de survie à long terme. Contre de l'Or, elle augmente le niveau de votre arme, recharge instantanément vos munitions au maximum et peut vous octroyer des bonus de puissance.

### ⚡ Power-ups Aléatoires
Les ennemis vaincus laissent parfois tomber des modules :
- **Aegis (Bouclier)** : Un champ de force qui repousse les zombies.
- **Chrono (Stase)** : Ralentit le temps pour vos ennemis, mais pas pour vous.
- **Instakill** : Vos balles éliminent n'importe quelle créature en un seul coup.
- **Double Damage & Rapid Fire** : Augmentez votre cadence et votre puissance de feu.

## 🎮 Commandes

| :--- | :--- |

| **Souris** | Regarder / Viser |

| **R** | Recharger l'arme |
| **1, 2, 3** / **Molette** | Sélectionner l'objet (Pistolet / Bombe / Pulse) |
| **O** | **Basculer entre le mode 2D et 3D** |
| **Maj (Shift)** | Sprinter |
| **Q** | Interagir avec la Punch Machine |
| **Echap** / **P** | Pause et Paramètres (Sensibilité, Volume) |
